###### Vuetify 설치

```bash
$ vue add vuetify
```

1. `Configure (advanced)`
2. `Use a pre-made template?`
   - Yes
3. `Use custom theme?`
   - No
4. `Use custom properties (CSS variables)?`
   - No
5. `Select icon font`
   - Font Awesome 5
6. `Use fonts as a dependency?`
   - No
7. `Use a-la-carte components?`
   - No
8. `Use babel/polyfill?`
   - Yes
9. `Select locale`
   - English

<br>

###### 프로젝트 실행

```bash
$ npm run serve
```

> Vuetify 로고로 변경되었다.
